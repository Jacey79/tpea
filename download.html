TPEA results

Per-evidence-type record-level data plus reference taxonomies. Each row in a raw evidence file corresponds to a single Analysis ID with all original fields preserved. Useful for re-analysis or ingesting into your own pipeline.

Dataset Description Download
Validated perturbation data Per-record functional perturbation evidence (KO / KD / overexpression / inhibitor / agonist). Includes Analysis ID, source, perturbation site, perturbation type and method, pain type and assessment, direction (Up / Down), effect, disease, and PMID. zip
Validated differential expression data Per-record hypothesis-driven RNA or protein-level confirmation (RT-qPCR, Western blot, IHC, IF, ELISA, etc.). Includes Analysis ID, source, expression assessment, direction, effect, disease, and PMID. zip
High-throughput profiling data Per-record multi-gene sequencing analyses (bulk RNA-Seq, miRNA-Seq, microarray). Includes Analysis ID, GEO accession, source, log2FC, p-value, adj. p, direction, and PMID. zip
Tissue / cell data Catalog of tissue and cell-line samples: standardized name, biological cell type, originating tissue, and anatomical site of derivation. Used to look up the source associated with each record's evidence. zip
Pain model hierarchical selection Selectable conditions for Pain model: hierarchical mapping of Pain model category to sub-category to source. For populating filter widgets. zip
Pain type hierarchical selection Selectable conditions for Pain type: hierarchical mapping of Pain type category to Pain assessment method, including assay descriptions. zip
Usage. All downloads are tab-separated value (TSV) files packaged in a single zip archive, suitable for direct import into R, pandas, or Excel. Redistribution is permitted under a CC-BY-4.0 license; please cite TPEA if used in published work (see Citation).
